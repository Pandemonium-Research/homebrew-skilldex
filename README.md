# homebrew-skilldex

Homebrew tap for [Skilldex](https://github.com/Pandemonium-Research/Skilldex) — a package manager and registry for Claude skill packages.

## Install

```bash
brew tap pandemonium-research/skilldex
brew install skilldex-cli
```

## Usage

```bash
skillpm --help
spm --help        # identical alias
```

## Update

```bash
brew upgrade skilldex-cli
```

## Uninstall

```bash
brew uninstall skilldex-cli
brew untap pandemonium-research/skilldex
```

## Requirements

- macOS or Linux
- Node.js 20+ (installed automatically as a dependency)
